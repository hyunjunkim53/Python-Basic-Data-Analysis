## Matplotlib 데이터 시각화 학습 기록

나도코딩 파이썬 활용편5(데이터 분석 및 시각화) 강의 실습 내용 정리

### 구성
* **01 ~ 07**: Matplotlib 기초 (그래프 기본, 축, 범례, 스타일, 파일 저장, 텍스트 등)
* **08 ~ 09**: 막대 그래프 (기본 및 심화)
* **10**: DataFrame 데이터를 활용한 시각화
* **11 ~ 12**: 누적 막대 및 다중 막대 그래프
* **13 ~ 14**: 원 그래프 (기본 및 심화 - 비율 표시, 범례 조절 등)
* **15 ~ 16**: 산점도 및 여러 그래프 동시 출력
* **17**: 종합 퀴즈 풀이 완료

### 요약
* `plt.figure(figsize=(x, y))`: 도화지(Canvas) 크기 설정 및 준비
* `plt.xticks(rotation=90)`: 축 눈금 이름이 겹치지 않도록 회전 설정
* `marker='o'`: 데이터 포인트의 위치를 시각적으로 강조
* `autopct='%.1f%%'`: 원 그래프(Pie chart)에서 각 조각의 비율을 소수점 첫째자리까지 표시
* `plt.legend(loc=(x, y))`: 범례의 위치를 그래프 외부 등 원하는 좌표로 세밀하게 조절
