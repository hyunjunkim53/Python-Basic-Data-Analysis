# Pandas 데이터 분석 학습 기록
나도코딩 파이썬 활용편5(데이터 분석 및 시각화) 강의 실습 내용 정리

## 구성
* **01 ~ 03**: Pandas 기초 (Series, DataFrame, Index)
* **04 ~ 05**: 파일 입출력 및 데이터 확인
* **06 ~ 09**: 데이터 선택 (loc, iloc, 조건 필터링 등)
* **10 ~ 14**: 데이터 전처리 (결측치 처리, 정렬, 그룹화 등)
* **15**: 종합 퀴즈 풀이 완료

## 요약
- `loc`(이름 기준)와 `iloc`(인덱스 숫자 기준) 차이점 정리
- 데이터 정렬 시 `ascending` 옵션 활용 (기본 오름차순)
- `groupby`를 통한 데이터 통계 추출 실습
